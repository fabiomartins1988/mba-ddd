# MBA Full Cycle - Domain Driven Design em GO

Projeto pratico de DDD desenvolvido durante o MBA FUll Cycle. A disciplina usa TypeScript/Nest.js como base; esta versao reimplementa o mesmo dominio (venda de ingresso) em Go, validando que os padroes taticos sao agnosticos a linguagem.

## Tecnologias

| Componente | Tecnologia |
|------------|------------|
| Linguagem  | GO 1.24+   |
